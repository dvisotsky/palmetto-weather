# Palmetto Weather — Frontend

React + TypeScript app for searching locations and displaying current weather and forecasts.

## Prerequisites

- Node.js 18+
- npm
- Backend running on `http://localhost:3000` (see `../backend/README.md`)

## Setup

```bash
npm install
```

## Running

```bash
npm run dev
```

The app opens at `http://localhost:5173`.

## Building

```bash
npm run build    # outputs to dist/
npm run preview  # preview the production build locally
```

## Testing

```bash
npm test           # run once
npm run test:watch # watch mode
```
